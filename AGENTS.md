# AGENTS.md

Shared guidance for automated coding agents working in this repository.

## Commit and PR messages

This is a public repository. Never add a `Co-Authored-By` trailer or a
session link to commit messages or pull request descriptions.
Write commit messages and PR descriptions as if authored solely by the
human contributor.

## Pull request descriptions

Use real Markdown newlines in pull request bodies. Do not pass escaped `\n`
sequences that render literally in GitHub.
